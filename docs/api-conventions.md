# API Conventions

- Versioned prefix: `/api/v1`
- ERP routes:
  - `/api/v1/erp/items`
  - `/api/v1/erp/purchase-orders`
  - `/api/v1/erp/suppliers`
- CRM routes:
  - `/api/v1/crm/contacts`
  - `/api/v1/crm/leads`
  - `/api/v1/crm/opportunities`
- Core auth routes:
  - `/api/v1/auth/login`
  - `/api/v1/auth/refresh`
  - `/api/v1/auth/me`
- Ops routes:
  - `/health`
  - `/api/v1/system/info`
