# sphincs ERP + CRM

Two separate web apps (ERP and CRM) backed by one modular API.

## Quick Start

```bash
pnpm install
pnpm dev
```

## Documentation

See [docs/index.md](docs/index.md).

## Repo Structure

- `apps/core-api`: NestJS backend
- `apps/erp-web`: ERP React app
- `apps/crm-web`: CRM React app
- `packages/*`: shared packages
